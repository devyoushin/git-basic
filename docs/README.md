# git-basic 문서

이 디렉토리는 주제별 번호 체계로 문서를 정리합니다. 앞 번호는 학습과 운영 흐름을 나타내며, 90번대는 저장소 관리용 문서입니다.

## 문서 구조

- [01-basics](./01-basics/): Git 기본기 - 초기화, clone, branch, remote, commit 기본 문서를 모읍니다.
- [02-workflow](./02-workflow/): Git 작업 흐름 - merge, rebase, stash, cherry-pick 문서를 모읍니다.
- [03-collaboration](./03-collaboration/): 협업 - 브랜치 전략, PR 워크플로우, 커밋 규칙 문서를 모읍니다.
- [04-history](./04-history/): 히스토리 탐색 - log, diff, blame, bisect, reset/revert 문서를 모읍니다.
- [05-advanced](./05-advanced/): 고급 기능 - hooks, reflog, submodule, worktree 문서를 모읍니다.
- [06-troubleshooting](./06-troubleshooting/): 문제 해결 - 충돌, 대용량 파일, 실수 복구 문서를 모읍니다.
- [07-cheatsheets](./07-cheatsheets/): 치트시트 - 자주 쓰는 명령과 긴급 대응 요약 문서를 모읍니다.
- [90-standards](./90-standards/): 문서 작성 표준 - 저장소 문서 작성 규칙과 운영 기준을 관리합니다.
- [91-templates](./91-templates/): 문서 템플릿 - 반복해서 사용하는 문서 템플릿을 관리합니다.
- [99-agents](./99-agents/): AI 에이전트 가이드 - Git 학습과 워크플로우 리뷰에 활용할 에이전트 역할 정의를 관리합니다.
