# Git Docs

Git 학습 문서는 주제별로 나눠 관리합니다.

| 폴더 | 내용 |
|------|------|
| `basics/` | Git 설정, init/clone, stage/commit, branch, remote |
| `workflow/` | merge, rebase, stash, cherry-pick |
| `history/` | log, diff, blame, bisect, reset/revert |
| `advanced/` | reflog, hooks, submodule, worktree |
| `collaboration/` | 브랜치 전략, 커밋 컨벤션, PR 워크플로우 |
| `troubleshooting/` | 충돌 해결, 실수 복구, 대용량 파일 문제 |
| `cheatsheets/` | 빠른 참조 문서 |
| `agents/` | AI 전문 에이전트 프롬프트 |
| `rules/` | 문서 작성과 Git 운영 규칙 |
| `templates/` | 커밋 메시지, gitignore 등 재사용 템플릿 |

실행 가능한 Git hook은 `../ops/README.md`를 참고합니다.
